Hello there! I often get asked about the tools and resources I use for my work. So, I've decided to create this page to list most stuff I use in my day-to-day life as a software developer. This includes most important items from my hardware setup, to the software tools I use. I hope you find it helpful!

## Software

- **[Arc Browser](https://arc.dev/)**: This is my primary browser.
- **[Obsidian](https://obsidian.md/)**: This is my primary note-taking app.
- **[Timing](https://timingapp.com/)**: This is my time tracking app.
- **[CleanShot X](https://cleanshot.com/)**: This is my screenshot app.
- **[Yabai](https://github.com/koekeishiya/yabai)**: This is my tiling window manager.

## Editors

- **[Neovim](https://neovim.io/)**: This is my primary editor. You can find the configuration in my [dotfiles](https://github.com/CuddlyBunion341/dotfiles/tree/main/.config/nvim).
- **[VSCode](https://code.visualstudio.com/)**: This is my backup editor with minimal plugins.
- **[Rubymine](https://www.jetbrains.com/ruby/)**: I used this during my internship.

## Terminal

- **[Alacritty](https://github.com/alacritty/alacritty)**: This is my primary terminal. You can find the configuration in my [dotfiles](https://github.com/CuddlyBunion341/dotfiles/blob/main/.config/alacritty/alacritty.toml).
- **[Warp](https://www.warp.dev/)**: I use this for its AI features.
- **[LazyGit](https://github.com/jesseduffield/lazygit)**: This is my git client. You can find the configuration in my [dotfiles](https://github.com/CuddlyBunion341/dotfiles/blob/main/lazygit/config.yml)
- **[Zsh](https://ohmyz.sh/)**: This is my shell. You can find the configuration in my [dotfiles](https://github.com/CuddlyBunion341/dotfiles/.zshrc)
- **[Tmux](https://github.com/tmux/tmux/wiki)**: This is my terminal multiplexer. You can find the configuration in my [dotfiles](https://github.com/CuddlyBunion341/dotfiles/blob/main/.tmux.conf)
- **[GitHub CLI](https://cli.github.com/)**: Used on a daily basis to initialize repositories, create PRs

## Gear

- **[Keychron Q3](https://www.keychron.com/products/keychron-q3-qmk-custom-mechanical-keyboard)**: This is my preferred keyboard.
- **[Asus PG32UQ](https://www.asus.com/Displays-Desktops/Monitors/All-series/ROG-SWIFT-PG32UQ/)**: This is a 4k, 120hz monitor.
- **[Audioengine A2+](https://audioengineusa.com/shop/wirelessspeakers/a2-wireless-computer-speakers/)**: These are my speakers.
- **[Secretlab Chair](https://www.secretlab.co/)**: This is my home office chair.
- **[M1 Pro MacBook Pro](https://www.apple.com/macbook-pro-14-and-16/)**: This is my primary development machine.
